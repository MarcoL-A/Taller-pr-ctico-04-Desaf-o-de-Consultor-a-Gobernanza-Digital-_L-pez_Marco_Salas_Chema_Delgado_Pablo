# Taller-práctico-04 Desafío de Consultor a Gobernanza Digital.
## López_Marco, Salas_Chema y Delgado_Pablo.

### El escenario:
Para una empresa de 25 empleados con presupuesto ajustado, la elección de SAP S/4HANA es una apuesta por la robustez y la escalabilidad a largo plazo. Aunque es la opción más costosa, se justifica si la organización planea un crecimiento acelerado o si requiere que su sistema de etiquetado y logística cumpla con estándares internacionales estrictos que SAP ya integra de forma nativa.
El cálculo del TCO a 3 años revela una inversión total de 175.900 €. Este gasto se divide principalmente en la suscripción anual (estimada en 55.500 € para los 25 usuarios), los costes operativos de infraestructura en la nube (1.800 € anuales para servicios de impresión y backups en AWS o Google Cloud) y vuestra inversión inicial en desarrollo de 4.000 €, destinada a la configuración personalizada de las etiquetas.
En conclusión, aunque el coste por usuario es elevado para una pyme, SAP garantiza que el software no se quedará pequeño. Sin embargo, para mantener el "presupuesto ajustado" mencionado, será vital que las 100 horas de desarrollo se gestionen con máxima eficiencia para evitar desviaciones en la implementación técnica.
